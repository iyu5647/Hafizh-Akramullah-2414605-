## Deskripsi Program
Program ini menggunakan library `Matplotlib` dan `Numpy` untuk mendemonstrasikan transformasi objek 3D.
Transformasi yang dilakukan meliputi:
1. **Objek Dasar:** Membuat kubus 3D dengan koordinat vertex awal.
2. **Translasi:** Memindahkan posisi kubus di ruang 3D.
3. **Rotasi:** Memutar kubus terhadap sumbu Z menggunakan matriks rotasi.

## Cara Menjalankan
1. Pastikan Python sudah terinstal.
2. Instal library pendukung: `pip install matplotlib numpy`.
3. Jalankan kode: `python uas_grafkom.py`.
